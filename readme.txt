0) Install appropriate VCP FTDI drivers.

1) Place the video in the data/movies. Name it "fingers.mp4".

2) Edit the "/extra files/fnkill.cmd". Write the seconds you  want to play the game instead of '10'.

3) Change the path of the appdata in the "/extra files/delete.cmd".

4) Configure the USB serial port to be COM8.

5) Place all files and folders in the "/extra files" into "C:/".

6) If game doesn't run, change the User Account Controls and set to minimal and also check if the QuickTime thing is giving errors, maybe fix the DLL.

7) Enjoy !!!
